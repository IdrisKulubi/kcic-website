# KCIC Website Administrator Training Manual

**Version:** 1.0  
**Date:** January 2026  
**Audience:** Communications Team, Content Editors, System Administrators  

---

## 1. Training Overview

### Objectives
By the end of this training, participants will be able to:
1.  Securely access and navigate the Admin Dashboard.
2.  Create, edit, and publish news, events, and success stories.
3.  Manage complex listings like Job Opportunities and Tenders.
4.  Update core site information (Team, Partners, Impact Stats).
5.  Handle confidential Whistleblower reports.

### Prerequisites
*   A computer with internet access.
*   Admin credentials (provided by the IT Lead).
*   Browser: Google Chrome, Firefox, or Edge (Latest versions).

## 3. Module Details

### Module 1: System Overview & Access
**Goal:** Familiarize users with the environment.

1.  **Login Procedure:**
    *   Navigate to `/admin`.
    *   Input Username/Password.
    *   *Security Note:* Never share passwords.
2.  **Dashboard Tour:**
    *   Explanation of the Sidebar Navigation.
    *   Understanding the "Sign Out" button.

### Module 2: Content Operations
**Goal:** Master daily updating tasks.

1.  **Managing News & Articles:**
    *   Difference between Categories (Press Release, Blog, Event).
    *   Using the Rich Text Editor (Bold, Italic, Links).
    *   **Exercise:** Create a draft article titled "Training Test" with a sample image.
2.  **Updating Hero Slides:**
    *   Importance of high-quality imagery.
    *   Adding a "Call to Action" button.
3.  **Impact Statistics:**
    *   How updating a number here reflects instantly on the homepage.

### Module 3: Advanced Management
**Goal:** Handle structured data and files.

1.  **Opportunities (Jobs & Procurement):**
    *   **Key Concept:** The "Type" selector determines where the post appears (Careers Page vs. Procurement Page).
    *   **Attachments:** How to upload PDF Terms of Reference (TORs).
    *   **Deadlines:** Setting auto-expiry dates for tenders.
    *   **Exercise:** Create a "Dummy Consultancy" post with a closing date of next week.
2.  **Programmes:**
    *   Editing the "About" text for GreenBiz or Agribiz.
    *   Updating the cover images.

### Module 4: Administration & Security
**Goal:** Maintain system integrity.

1.  **Team Management:**
    *   Adding new staff members or board directors.
    *   Reordering profiles (if applicable).
2.  **Whistleblower Reports:**
    *   **Confidentiality Protocol:** Who is allowed to view these?
    *   Processing a report: Changing status from "New" to "Investigating".
3.  **Troubleshooting:**
    *   What to do if an image doesn't load (Check file size/format).
    *   Who to contact for technical support.

## 4. Cheat Sheet: Do's and Don'ts

| Do's | Don'ts |
| :--- | :--- |
| **DO** compress images before uploading to keep the site fast. | **DON'T** upload images larger than 2MB. |
| **DO** use clear, descriptive titles for all posts. | **DON'T** copy-paste text directly from Word without clearing formatting. |
| **DO** log out when using a shared computer. | **DON'T** share your admin login details with interns or unauthorized staff. |
| **DO** double-check links after publishing. | **DON'T** delete immediate items without backing up content. |
